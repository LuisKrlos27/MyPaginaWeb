Conocimiento

Cuento con un conocimiento básico en una variedad de herramientas y tecnologías que abarcan el campo de la Ingeniería en Sistemas:

- Programación: Tengo experiencia con lenguajes de programación como Java, JavaScript, Python y C++. He utilizado entornos de desarrollo como NetBeans y Visual Studio Code (VSC) para desarrollar aplicaciones y proyectos.

- Control de Versiones: Estoy familiarizado con sistemas de control de versiones como Git y plataformas de alojamiento como GitHub y Bitbucket. Utilizo estos sistemas para gestionar y colaborar en proyectos de desarrollo de software.

- Diseño Web: He trabajado con tecnologías web como HTML y CSS, y he utilizado el framework Bootstrap para crear sitios web atractivos y receptivos.

- Sistemas Operativos: Tengo experiencia en el uso de sistemas operativos como Windows, Linux (incluyendo distribuciones como Kali Linux y Debian) y estoy familiarizado con sus características y comandos básicos.

- Cableado Estructurado: Cuento con conocimientos básicos en cableado estructurado, lo que incluye la planificación y el despliegue de infraestructuras de red.

- Autocad: He utilizado AutoCAD para crear y editar planos y diseños técnicos en el contexto de proyectos de ingeniería.

- Redes: Tengo un conocimiento fundamental en conceptos de redes, incluyendo configuración, topologías y protocolos de comunicación.

- Base de Datos: Poseo conocimientos básicos en el diseño y uso de bases de datos.

- PostgreSQL: Estoy familiarizado con PostgreSQL, un sistema de gestión de bases de datos relacional, y su uso en aplicaciones y proyectos.

- Modelo Cliente-Servidor: Comprendo el modelo cliente-servidor y su aplicación en el desarrollo de aplicaciones y sistemas distribuidos.

Estos son algunos de mis conocimientos básicos en el campo de la Ingeniería en Sistemas. Estoy emocionado por seguir aprendiendo y expandiendo mi experiencia en estas áreas, así como por aplicar mis habilidades en proyectos futuros.
